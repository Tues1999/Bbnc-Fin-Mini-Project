## 1. ความต้องการของระบบ (System Requirements)

เพื่อให้ระบบสารสนเทศทำงานได้อย่างมีประสิทธิภาพและเสถียรภาพ เครื่องคอมพิวเตอร์ของผู้หน้างานควรมีคุณสมบัติพื้นฐานดังนี้:

*   **ระบบปฏิบัติการ:** Windows 10 หรือ Windows 11
*   **ซอฟต์แวร์พื้นฐาน:** โปรแกรม Python เวอร์ชัน 3.8 ขึ้นไป (จำเป็นสำหรับการประมวลผลทางด้านเซิร์ฟเวอร์)
*   **โปรแกรมเว็บเบราว์เซอร์:** Google Chrome, Microsoft Edge หรือ Mozilla Firefox เวอร์ชันปัจจุบัน

## 2. ขั้นตอนการติดตั้ง (Installation)

กรุณาดำเนินการตามขั้นตอนดังต่อไปนี้เพื่อติดตั้งระบบลงในเครื่องคอมพิวเตอร์:

### ขั้นตอนที่ 1: การเตรียมความพร้อมทางด้านซอฟต์แวร์
1.  ดาวน์โหลดโปรแกรม Python รุ่นล่าสุดได้ที่เว็บไซต์ทางการของ Python (https://www.python.org/downloads/)
2.  ดำเนินการติดตั้งโปรแกรมลงในเครื่องคอมพิวเตอร์
    *   **ข้อควรระวัง:** ในหน้าต่างการติดตั้ง ก่อนกดปุ่ม Install กรุณาทำเครื่องหมายถูกที่ช่อง **"Add Python to PATH"** เพื่อให้ระบบปฏิบัติการสามารถเรียกใช้คำสั่ง Python ได้อย่างถูกต้อง

### ขั้นตอนที่ 2: การติดตั้งตัวระบบ
1.  คัดลอกโฟลเดอร์รหัสต้นฉบับ `ban_bang_nam_chuet_financial` ไปยังตำแหน่งที่ต้องการจัดเก็บ (เช่น Desktop หรือ Drive D)
2.  เข้าสู่ภายในโฟลเดอร์ดังกล่าว

### ขั้นตอนที่ 3: การเริ่มต้นระบบ (System Startup)
1.  ค้นหาไฟล์ประเภท Batch File ชื่อว่า `start_server.bat`
2.  ดับเบิ้ลคลิกที่ไฟล์ดังกล่าวเพื่อเริ่มการทำงานของระบบ
3.  ระบบจะดำเนินการตรวจสอบความพร้อม สร้างสภาพแวดล้อมจำลอง (Virtual Environment) และติดตั้งชุดคำสั่งที่จำเป็นโดยอัตโนมัติ
4.  เมื่อกระบวนการเสร็จสมบูรณ์ ระบบจะเปิดหน้าต่างเว็บเบราว์เซอร์เข้าสู่ระบบให้โดยอัตโนมัติ
    *   **หมายเหตุ:** โปรดอย่าปิดหน้าต่าง Command Prompt (หน้าจอสีดำ) ในขณะที่กำลังใช้งานระบบ เนื่องจากเป็นการปิดการทำงานของเซิร์ฟเวอร์

## 3. คู่มือการใช้งานแยกตามบทบาท (User Manual)

### 3.1 สำหรับบุคลากรครู (Teacher)
**ขอบเขตหน้าที่:** บันทึกและนำส่งคำร้องขอเบิกงบประมาณ

1.  **การเข้าสู่ระบบ:** ลงชื่อเข้าใช้ด้วยบัญชีผู้ใช้ หรือลงทะเบียนสมาชิกใหม่หากยังไม่มีบัญชี
2.  **การส่งคำร้อง:**
    - เลือกเมนู "ส่งคำร้อง" จากแถบเมนู
    - ระบุรายละเอียดของรายการ ได้แก่ วันที่, ประเภทบัญชี, หมวดหมู่รายจ่าย, และจำนวนเงิน
    - ตรวจสอบความถูกต้องและบันทึกข้อมูลเพื่อนำส่งเข้าสู่ระบบตรวจสอบ
3.  **การติดตามสถานะ:**
    - เลือกเมนู "สถานะคำขอ" เพื่อตรวจสอบความคืบหน้าของการพิจารณาอนุมัติจากฝ่ายการเงินและฝ่ายบริหาร

### 3.2 สำหรับเจ้าหน้าที่การเงิน (Finance Officer)
**ขอบเขตหน้าที่:** ตรวจสอบความถูกต้องของคำร้อง และบริหารจัดการทะเบียนคุมงบประมาณ

1.  **การอนุมัติคำร้อง:**
    - เลือกเมนู "อนุมัติ"
    - ตรวจสอบรายละเอียดรายการคำร้องที่ส่งเข้ามา
    - หากข้อมูลถูกต้อง ให้ดำเนินการกดปุ่ม "การเงินอนุมัติ" (ปุ่มสีเหลือง)
2.  **การบริหารทะเบียนคุม:**
    - เข้าถึงทะเบียนคุมตามประเภทงบประมาณ (เงินอุดหนุน / เงินรายได้สถานศึกษา / เงินอาหารกลางวัน)
    - บันทึกรายการรายรับ-รายจ่ายเพิ่มเติม หรือแก้ไขรายการที่ผิดพลาด
    - ระบบจะทำการบันทึกประวัติการแก้ไข (Audit Trail) เพื่อความโปร่งใสและตรวจสอบได้
3.  **การออกรายงาน:**
    - เลือกเมนู "Export to Excel" เพื่อดาวน์โหลดรายงานสรุปประจำเดือนในรูปแบบไฟล์ Excel

### 3.3 สำหรับผู้อำนวยการสถานศึกษา (Director)
**ขอบเขตหน้าที่:** อนุมัติการเบิกจ่ายขั้นสุดท้าย และกำกับดูแลภาพรวมงบประมาณ

1.  **การอนุมัติคำร้อง:**
    - เลือกเมนู "อนุมัติ" เพื่อดูรายการที่ผ่านการตรวจสอบจากเจ้าหน้าที่การเงินแล้ว
    - ดำเนินการกดปุ่ม "ผอ. อนุมัติ" (ปุ่มสีเขียว)
    - **ผลลัพธ์:** เมื่อได้รับการอนุมัติขั้นสุดท้าย ระบบจะดำเนินการบันทึกรายการลงในสมุดบัญชีทะเบียนคุมโดยอัตโนมัติทันที
2.  **การตรวจสอบ:**
    - สามารถเข้าดูและแก้ไขทะเบียนคุมได้เช่นเดียวกับเจ้าหน้าที่การเงิน
    - สามารถตรวจสอบประวัติการแก้ไขข้อมูลย้อนหลังได้

## 4. การแก้ไขปัญหาเบื้องต้น (Troubleshooting)

**กรณีเปิดไฟล์ `start_server.bat` แล้วโปรแกรมปิดตัวลงทันที**
สันนิษฐานว่ายังไม่ได้ติดตั้งโปรแกรม Python หรือการตั้งค่า Path ไม่สมบูรณ์ กรุณาดำเนินการติดตั้ง Python ใหม่อีกครั้ง และตรวจสอบการทำเครื่องหมายที่ช่อง "Add Python to PATH" ระหว่างการติดตั้ง

**กรณีเว็บไซต์ไม่ปรากฏขึ้นมาอัตโนมัติ**
ให้เปิดโปรแกรมเว็บเบราว์เซอร์ และพิมพ์ที่อยู่ในช่อง Address Bar ดังนี้: `http://127.0.0.1:5000`

## 5. ข้อมูลบัญชีสำหรับการทดสอบระบบ (Default Credentials)

**คำเตือน:** ชุดข้อมูลบัญชีผู้ใช้นี้จัดทำขึ้นเพื่อวัตถุประสงค์ในการทดสอบระบบและเรียนรู้วิธีการใช้งานเบื้องต้นเท่านั้น เมื่อนำระบบไปใช้งานจริง **ฝ่ายผู้ดูแลระบบควรดำเนินการเปลี่ยนรหัสผ่านหรือลบข้อมูลผู้ใช้งานชุดนี้ออก** เพื่อความปลอดภัยของข้อมูล

| บทบาท (Role) | ชื่อผู้ใช้งาน (Username) | รหัสผ่าน (Password) |
| :--- | :--- | :--- |
| **คุณครู (Teacher)** | `teacher` | `pass1234` |
| **เจ้าหน้าที่การเงิน (Finance)** | `finance` | `pass1234` |
| **ผู้อำนวยการ (Director)** | `director` | `pass1234` |
